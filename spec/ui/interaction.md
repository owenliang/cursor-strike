# 交互流程文档

## 全局流程

```
首页登录 → 游戏主界面 → 训练结束评估
  ↑                              ↓
  └──── 返回主页 ←── 再来一局 ←──┘
```

三个页面顺序流转，无分支跳转。评估页可选择"再来一局"回到游戏，或"返回主页"回到登录。

---

## Slide 1 — 登录首页

### 入口行为
- 页面加载后自动聚焦到昵称输入框
- 背景百分比网格 + 柔光静态展示，无动画

### 输入框交互
- placeholder："请输入游戏昵称..."，placeholder颜色 `#888888`
- 输入时文字为深色 `#1a1a1a`
- 最大 12 字符限制
- 输入框为空时点击"开始游戏"按钮无效，输入框抖动提示（shake 动画 400ms）
- 回车键也可触发开始游戏

### 开始游戏按钮
- 点击后：
  1. 校验昵称非空
  2. 存储昵称至状态
  3. 过渡动画 → 进入游戏主界面
- 按钮文字/图标为白色（`#ffffff`）

### 页面过渡
- 登录 → 游戏：当前页淡出（opacity 1→0，200ms），新页淡入
- 切换期间禁止交互

## Slide 2 — 游戏主界面

### 游戏初始化
- 进入游戏页后显示 3 秒倒计时（"3 → 2 → 1 → GO"），居中大字 Anton 字体霓虹绿
- 倒计时结束后开始正式计时，小球开始生成

### 视差背景
- 游戏背景随鼠标移动产生视差偏移（0.5倍幅度）
- 鼠标移右 → 背景偏右，鼠标移左 → 背景偏左
- 通过 CSS transform translate 实现

### 核心游戏机制

#### 目标小球生成规则
- 小球在**屏幕中心周边 30–80px 范围内**随机位置刷新（anchor 为游戏区域中心坐标，非鼠标位置）
- 小球之间互不重叠（最小间距 ≥ 小球半径 + 现有小球半径 + 6px gap）
- 生成节奏：每隔 **280ms** 恒定速度持续刷新，不加速也不减速
- 画面同时最多存在 **15** 颗小球
- 小球尺寸**固定**：半径 18px（直径 36px）
- 小球出现时有缩放入场动画：ballSpawn（scale 0→1, opacity 0→1）

#### 目标小球外观
- 填充径向渐变 `#22c55e → #16a34a`
- boxShadow 发光效果：`0 0 10px rgba(34,197,94,0.19), 0 2px 4px rgba(0,0,0,0.08)`
- 无悬停态变化

#### 点击小球（命中）
- 小球被点击后：
  1. 标记 dying 状态，缩放消失动画 ballDie（scale 1→0, opacity 1→0, 200ms）
  2. 200ms 后从列表移除
  3. 在点击位置产生霓虹绿粒子扩散效果（5个小点向四周扩散，hitParticle 300ms）
  4. 得分 +1，命中次数 +1
  5. 记录反应时间 = 点击时刻 - 小球生成时刻
  6. 更新 HUD 命中和命中率数值

#### 点击空白区域（失误）
- 点击非小球区域：
  1. 准星短暂闪红（crosshairFlash 动画 150ms），颜色 `#ef4444`
  2. 失误次数 +1
  3. 更新 HUD 失误和命中率数值

#### 命中率计算
```
命中率 = 命中次数 / (命中次数 + 失误次数) × 100%
```
实时计算，每次点击后即时更新

### HUD 面板

#### 右上角数据面板
- 三行数据，每行带 Tip 悬浮解释（lucide info 图标）
- **命中**：显示为 hits/totalTargets 格式（如 "8/15"），`#22c55e`；Tip："成功点击数 / 刷新目标总数"
- **失误**：显示失误次数，红色 `#ef4444`；Tip："点击空白区域的次数"
- **剩余时间**：默认 30 秒倒计时，`#1a1a1a` 显示；Tip："倒计时归零后训练结束"
  - 最后 10 秒文字变红 `#ef4444` + timeFlash 闪烁动画
- 面板底色 `rgba(245,245,245,0.88)`，边框 `#e5e5e5`

#### 左上角控制按钮
- **暂停**：点击后游戏暂停
  - 小球停止生成，计时暂停
  - 显示暂停蒙层
- **退出**：点击后回到登录首页，当前数据丢弃
- 按钮底色 `rgba(240,240,240,0.5)`，边框 `#e5e5e5`，文字/图标 `#666666`

### 游戏结束条件
- 30 秒倒计时归零 → 自动进入训练结束评估页
- 过渡动画：淡出+淡入 200ms

## Slide 3 — 训练结束评估

### 数据计算规则

| 指标 | 计算方式 | 显示格式 | Tip 解释 |
|------|----------|----------|----------|
| 总得分 | 命中次数 × 1 | 整数 | 每命中1个目标得1分，总得分 = 命中次数 |
| 命中 | 命中点击数 / 刷新目标总数 | hits/total | 成功点击的目标数 / 累计刷新的目标总数 |
| 失误 | 累计空白点击数 | 整数（红） | 点击空白区域（未命中任何目标）的次数 |
| 命中率 | 命中 / (命中 + 失误) × 100% | 百分比，1位小数 | 成功点击次数 ÷ (成功点击 + 失误点击) × 100% |
| 平均反应用时 | 所有命中球的反应时间平均值 | 秒，2位小数 | 从目标出现到成功点击的平均间隔时间（秒） |
| 每秒命中 | 总命中次数 ÷ 30 | 小数，1位 | 总命中次数 ÷ 游戏时长(秒)，衡量击靶手速 |

### 页面入场动画
- 卡片逐张淡入+上移（fadeInUp 400ms + stagger delay）
- 每张间隔 80ms

### 再来一局按钮
- 点击后：
  1. 清空本次数据
  2. 保持昵称不变
  3. 过渡进入游戏主界面，重新开始
- 按钮底色 `#22c55e`，文字/图标白色 `#ffffff`

### 返回主页按钮
- 点击后：
  1. 过渡回到登录首页
- 按钮底色 `#f0f0f0`，边框 `#e5e5e5`，文字/图标 `#666666`

### 数据持久化
- 每局结束将结果存入 localStorage，键 `cursor_strike_history`
- 保留最近 20 局记录
- 评估页暂不展示历史，后续可扩展
