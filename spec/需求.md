# Cursor Strike — FPS 鼠标手速训练游戏

电脑端 Web 版 FPS 鼠标手速训练游戏，一共三个页面，深灰碳色+霓虹绿高光电竞风格。

1. 登录首页界面
顶部游戏英文大字标题 "CURSOR STRIKE"，中间有昵称输入框，提示请输入游戏昵称，下方有醒目开始游戏按钮；背景用低调科技网格暗纹+微弱径向柔光，界面简洁电竞风，只保留必要输入和按钮元素。输入回车也可进入游戏。

2. 游戏主界面（核心玩法）
全屏深灰碳色纯净背景（#1a1a2e），背景随鼠标轻微视差偏移（0.5倍幅度）；
**小球在屏幕中心周边约30~80像素范围内随机刷新，小球之间互不重叠（最小间距 ≥ 小球直径 + 6px）**；
小球固定大小（半径18px），以280ms间隔持续稳定刷新，画面同时最多15个小球；
玩家鼠标隐藏，由自定义FPS十字准星跟随鼠标（8px短臂+4px中心间隙+4px中心点）；
右上角HUD面板：命中(hits/total)、失误、剩余时间，每个指标带Tip悬浮解释；
左上角：暂停/退出按钮；
小球用霓虹绿填充+轻微外发光，被点击后缩放消失动画+粒子扩散效果；
点击空白区域准星闪红（失误），全程无多余遮挡元素。

3. 训练结束评估界面
大标题："训练完成"
居中数据卡片布局，展示6项评估指标：
总得分、命中(hits/totalTargets)、失误、命中率、平均反应用时、每秒命中；
每个指标带Tip悬浮解释（计算口径描述）；
底部两个按钮：再来一局、返回主页；
关键数据数字用霓虹绿高亮，失误用红色，卡片半透明磨砂深色风格，排版整洁专业。

游戏时长：30秒
计分规则：每命中1个目标得1分（总得分 = 命中次数）
命中率 = 成功点击次数 ÷ (成功点击 + 失误点击) × 100%
每秒命中 = 总命中次数 ÷ 游戏时长(秒)

整体要求：
电脑端web端适配，极简FPS练靶风格，无移动端控件，配色统一深灰碳(#1a1a2e)+霓虹绿(#4ade80)，按钮、输入框保持统一圆角科技质感。
